# Git-tiimiprojekti – Template

Tämä projekti toimii lähtöpohjana Git-tiimiprojektille.

Projektin tavoitteena on harjoitella:
- Git-versionhallintaa
- GitHubin käyttöä
- kehityshaarojen käyttöä
- tiimityöskentelyä
- HTML-, CSS- ja JavaScript-kehitystä

Projektin aiheena on “Meidän tiimi” -sivusto.

---

## Projektin rakenne

```text
projekti/
│
├── index.html
├── pages/
│   └── opiskelija_malli.html
│
├── css/
│   └── style.css
│
├── js/
│   └── script.js
│
└── images/
```

---

## Projektin idea

Projektissa toteutetaan:
- yhteinen etusivu
- jokaiselle opiskelijalle oma profiilisivu
- yhteinen ulkoasu ja navigaatio
- mahdollisia uusia ominaisuuksia projektin jatkokehitysvaiheessa

---

## Työskentelyohjeita

### Kehityshaarat

Jokainen opiskelija:
- luo oman kehityshaaran
- tekee muutokset omaan branchiinsa
- tekee committeja työskentelyn aikana
- yhdistää oman branchinsa takaisin päähaaraan

---

## Mallisivun käyttö

`opiskelija_malli.html` toimii pohjana opiskelijoiden omille sivuille.

Voit:
- kopioida tiedoston uudella nimellä
- muokata sisältöä
- lisätä oman kuvan
- päivittää navigaation

---

## Kuvien lisääminen

Lisää kuvat `images`-kansioon.

Projektissa käytetään esimerkiksi:
- yhteistä tiimikuvaa
- opiskelijoiden profiilikuvia

---

## Sivuston muokkaaminen

Projektin ulkoasua ja sisältöä saa muokata vapaasti.

Voitte esimerkiksi:
- vaihtaa väriteemaa
- lisätä animaatioita
- toteuttaa uusia JavaScript-toimintoja
- tehdä sivustosta oman tiimin näköisen

---

## Projektin käynnistäminen

1. Kloonaa repository omalle koneelle

```bash
git clone REPOSITORY_URL
```

2. Avaa projektikansio

3. Käynnistä `index.html` selaimessa

---

## Muistettavaa

- Tee committeja riittävän usein
- Kirjoita selkeitä commit-viestejä
- Hae tarvittaessa uusimmat muutokset ennen mergeä
- Testaa sivuston toimivuus mergejen jälkeen
